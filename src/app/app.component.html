Bom dia, James. Seja bem vindo.
Em Python não existem arrays e sim listas. Já que são dinâmicas, você não precisa definir um tamanho para elas nem o tipo.

Ex: lista = []

Depois você só precisa inserir dados na lista. Recomendo usar o método append, que sempre irá colocar dados no último index da lista.
